The 4-Year Master Plan

A step-by-step framework to navigate your BTech CSE degree. No confusion, just pure execution.

Y1

ভিত্তি গড়ো (Foundation)

Semester 1

  • C Programming — loops, functions, arrays, pointers
  • Basic Math — Discrete Mathematics, Logic Gates
  • Computer Fundamentals — OS basics, how internet works
  • Typing speed বাড়াও (40+ WPM)

Semester 2

  • Python basics — C জানলে Python সহজ লাগবে
  • Git & GitHub — account খোলো, daily use করো
  • OOP concepts — Java বা C++ দিয়ে শেখো
  • ছোট project বানাও (calculator, quiz app, to-do list)
🎯 Year 1 Target:
  • একটা language ভালোভাবে জানবে
  • GitHub এ 2-3টা project থাকবে
  • কোন field ভালো লাগে সেটা বুঝে যাবে
Y2

দক্ষতা বাড়াও (Skill Building)

Semester 3

  • DSA (Data Structures & Algorithms) — সবচেয়ে গুরুত্বপূর্ণ বছর
  • Array, LinkedList, Stack, Queue, Tree, Graph
  • Database — MySQL, basic SQL queries
  • Linux/Terminal — commands শেখো
  • LeetCode এ Easy problems শুরু করো

Semester 4

  • DSA continue — Sorting, Searching, Recursion, DP
  • একটা Domain বেছে নাও:
    • Web: HTML, CSS, JS, React
    • App: Flutter বা Kotlin
    • AI/ML: NumPy, Pandas
    • DevOps: AWS, Docker
🎯 Year 2 Target:
  • DSA এর core topics শেষ
  • একটা domain এ কাজ শুরু হয়েছে
  • LinkedIn profile বানাও & GitHub active
Y3

Industry Ready (Real World)

Semester 5

  • Chosen domain এ deep dive করো (Full stack, ML projects, Apps)
  • System Design basics শেখো
  • Open Source Contribution শুরু করো
  • Internship খোঁজা শুরু করো

Semester 6

  • Internship — যেকোনো একটা পেলেই করো
  • Portfolio website বানাও
  • 2-3টা বড় project বানাও resume এর জন্য
  • LeetCode Hard problems & FAANG prep
🎯 Year 3 Target:
  • কমপক্ষে 1টা internship করা
  • 3-4টা solid project & Resume ready
  • 100+ LeetCode solved
Y4

Placement & Future

Semester 7

  • Campus Placement Prep: DSA revision, Mock interviews
  • Higher Study (MS/MTech): GATE/GRE prep, Research papers
  • Final Year Project শুরু করো

Semester 8

  • Final Year Project শেষ করো ও deploy করো
  • Placement/Offers handle করো
  • Certificates নাও (AWS, Meta, Google)
🎯 Year 4 Target:
  • Job offer বা MS admission
  • Strong portfolio & clear career direction
